## 중요한 의사결정

- **이메일 추상화**: `EmailService` 인터페이스 + 프로파일별 빈. dev 기본은 `ConsoleEmailService`(stdout dump) — SMTP 의존 없이 RED→GREEN 가능. 사용자 승인 2026-05-02.
- **토큰 저장**: 평문 토큰은 이메일/링크에만, DB에는 SHA-256 해시(`token_hash`)만 저장. token 노출 시 직접 reset 불가능.
- **TTL 30분**, **1회용**(`used_at`).
- **anti-enumeration**: `forgot`은 가입/미가입 모두 200. 토큰 INSERT 분기만 다르고 응답은 동일.
- **reset 시 모든 세션 무효화** / **change 시 다른 세션만 무효화**.
- **PW 정책 강화 트랙은 별도**. 본 트랙은 SignupService 기존 정책을 `PasswordPolicyValidator`로 추출만 하고 강화는 다음 트랙에서.
